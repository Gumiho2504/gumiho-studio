# 🦊 Gumiho Studio — Portfolio Website

> An indie game studio portfolio built with **Vue 3 (Composition API)** + **Tailwind CSS 3** + **Vue Router 4**

## Quick Start

```bash
npm install
npm run dev       # http://localhost:5173
npm run build
npm run preview
```

## Project Structure

```
src/
├── assets/css/main.css       # Global styles + CSS variables
├── components/
│   ├── FoxLogo.vue            # SVG 9-tailed fox mascot
│   ├── GameCard.vue           # Reusable game card
│   ├── LightboxModal.vue      # Screenshot lightbox
│   ├── NavBar.vue             # Fixed navbar + mobile menu
│   ├── AppFooter.vue          # Footer
│   └── ParticleCanvas.vue     # Hero particle network
├── data/games.js              # All game data
├── router/index.js            # Vue Router 4
└── views/
    ├── HomeView.vue
    ├── GamesView.vue
    ├── GameDetailView.vue
    ├── AboutView.vue
    └── ContactView.vue
```

## Tech Stack
- Vue 3 (Composition API + script setup)
- Vue Router 4 (hash history)
- Tailwind CSS 3
- Vite
